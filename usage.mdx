Using Teams but (actually) good is really simple, it works the same way on your browser and on the app

Firstly open the app and connect yourself to your account

<Note>
  Keep in mind the info part in [desktop](/desktop), some issues might be written on it
</Note>

Next to your avatar, press the `3 dots`, and press on `Settings` (or whatever it's called in your language)

<Frame>
  ![Teams Settings 3 Dots](/images/teams-settings-3-dots.jpg)
</Frame>

Now select **Teams but (actually) good Settings** in the settings tab

<Frame>
  ![Teams Settings List](/images/teams-settings-list.jpg)
</Frame>

Once selected you'll see this, this is the plugin list

<Info>
  Plugins get added / removed with time, so your plugin list might not look like this one (more plugin will probably be added)
</Info>

<Frame>
  ![Tbag Plugin List](/images/tbag-plugin-list.jpg)
</Frame>

<Info>
  Some plugins / settings in plugins / themes, might require you to restart teams, a big button will appear if that's the case, you just need to press on it, like the one below

  <Frame>
    ![Tbag Restart Button](/images/tbag-restart-button.jpg)
  </Frame>
</Info>

Here you can enable or disable plugins (beside the enable by default ones, like `Settings` and `Telemetry` which are mandatory)

Plugins can also have settings, like the `Settings` plugin which you can disable or enable the Themes

<Frame>
  ![Tbag Settings Setting](/images/tbag-settings-setting.jpg)
</Frame>

When Themes are enable (which is by default) you'll see this under the plugin list

<Info>
  Like for plugins, Themes get added / removed with time, so you'll probably see something different than the screenshot
</Info>

<Frame>
  ![Tbag Theme List](/images/tbag-theme-list.jpg)
</Frame>

Then you can select the theme you want and restart themes to apply the changes.
