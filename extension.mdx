Teams but (actually) good works on Chrome based browser and Firefox based browser, so most of the browser are supported by the extension.

| Browser | Tested ? | Info |
| --- | --- | --- |
| Chrome | <Icon icon="check" /> | - |
| Firefox | <Icon icon="check" /> | Need version 128.0 or above due to manifest V3 compatibility and how the extension works |
| Zen (Firefox) | <Icon icon="check" /> | - |
| Safari | <Icon icon="check" /> | Need a Mac that uses Apple Silicon (M1 - MX) Intel Mac are not supported |
| Edge | <Icon icon="check" /> | - |

To download the extension, heads up on the [website](https://teamsbutactuallygood.dev/download) and select your browser type, you'll be redirected to the store of your browser (Chrome web store or Firefox Add-ons) to download the extension, or you'll get the file to download it if not available on official stores.
