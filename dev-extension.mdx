Here you'll have all the information you need to be able develop the with the extension

Firstly you'll need to download [Chrome](https://www.google.com/intl/fr/chrome/)

<Info>
  Any other Chrome based browser will work, Firefox is not supported at this time
</Info>

Now in your terminal run this command

```bash
bun run watch
```

After running it you should see Chrome open, that's a good sign, now go to the Settings and Extension like the screenshot under

<Frame>
  ![Chrome Extension](/images/chrome-extension..jpg)
</Frame>

Now enable the dev mod

<Frame>
  ![Chrome Devmod Extension](/images/chrome-devmod-extension.jpg)
</Frame>

And press the "Load unpacked extension", navigate to the extension path and open it, it should look like that

<Frame>
  ![Chrome Extension List](/images/chrome-extension-list.jpg)
</Frame>

Now open the extension and enable the "Developer Mode"

<Frame>
  ![Extension Open](/images/extension-open.jpg)
</Frame>

Now go on [Teams](https://teams.microsoft.com) and you should be able to see Teams but (actually) good in the settings like showed in [Usage](/usage)

<Tip>
  Don't forget to install [React DevTools](https://chromewebstore.google.com/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi) to work on plugins
</Tip>

Go check Plugis and Themes to understand how they work to make your own one
